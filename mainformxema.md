# Главная форма Xema

Главная форма версии Xema — это основной интерфейс приложения для управления анализами и устройством анализатора.

## Интерфейс

- **Кнопка "Выбрать"**: Создает новый анализ. После нажатия выберите шаблон и начните процесс.
- **Кнопка "Поиск"**: Открывает форму поиска существующих результатов анализов.
- **Кнопка "Экспорт"**: Экспортирует отчет о текущем анализе в файл (PDF или другой формат).
- **Кнопка "Открыть лоток"**: Открывает лоток устройства для размещения пластины.
- **Кнопка "Закрыть лоток"**: Закрывает лоток устройства перед началом сканирования.
- **Кнопка "Настройки"**: Открывает окно настроек для конфигурации устройства и приложения.

## Как начать новый анализ

1. Убедитесь, что устройство подключено и настроено.
2. Нажмите кнопку **"Выбрать"**.
3. Выберите подходящий шаблон анализа из списка.
4. Нажмите **"Открыть лоток"**, если лоток закрыт.
5. Поместите пластину с образцами в лоток.
6. Нажмите **"Закрыть лоток"**.
7. Дождитесь завершения сканирования и просмотрите результаты.

## Просмотр существующих результатов

1. Нажмите кнопку **"Поиск"**.
2. В форме поиска укажите критерии (даты, тип анализа и т.д.).
3. Выберите нужный результат и нажмите **"OK"** для просмотра.

## Экспорт отчета

1. После завершения анализа нажмите **"Экспорт"**.
2. Выберите формат файла и место сохранения.
3. Отчет будет содержать данные, графики и статистику.

## Управление устройством

- Используйте кнопки **"Открыть лоток"** и **"Закрыть лоток"** для доступа к пластине.
- Если кнопки неактивны, проверьте подключение устройства в настройках.

Для дополнительных настроек нажмите **"Настройки"**.
