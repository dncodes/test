# Просмотр результатов анализа

Форма просмотра результатов анализа позволяет детально изучить данные сканирования, просмотреть графики, статистику и экспортировать отчеты.

## Интерфейс

- **Таблица платы:** Визуализация 96-луночной платы с оптическими плотностями (OD) по лункам (строки A-H, колонки 1-12).
- **Таблица детальных результатов:** Список с колонками: Лунка (Well), Имя образца (CaseName), OD, Индекс, Титр (Titer), Группа (Group), Результат (Result).
- **Выпадающий список групп:** Выберите группу для фильтрации данных. При изменении обновляются таблицы и графики.
- **Кнопки навигации:** "Предыдущая группа" и "Следующая группа" для переключения между группами.
- **Кнопка "Перечитать":** Обновляет данные анализа.
- **Кнопка "Печать":** Экспортирует отчет в PDF или Excel с таблицами и графиками.
- **Кнопка "Сохранить":** Сохраняет текущие результаты в базу данных.
- **Кнопка "Сохранить все":** Экспортирует все данные анализа в файл.
- **График распределения:** Гистограмма титров по группам.
- **График истории:** Линии максимальных, минимальных, средних титров и коэффициента вариации (CV) по возрастам.
- **Радиокнопки для графика истории:** Выберите тип данных: максимальные титры, минимальные, средние, геометрическое среднее или CV.
- **Таблица статистики группы:** Сводка по выбранной группе (число образцов, возраст, дата, статистические показатели).

## Как работать с результатами анализа

1. Откройте форму результатов, выбрав анализ из поиска или после завершения сканирования.
2. Изучите таблицу платы для быстрого обзора OD значений по лункам.
3. В детальной таблице просмотрите данные по каждому образцу: титры, группы и результаты.
4. Выберите конкретную группу из выпадающего списка для фокусировки на ней — таблицы и графики обновятся автоматически.
5. Используйте кнопки **"Предыдущая группа"** и **"Следующая группа"** для быстрого переключения между группами.
6. Просмотрите **график распределения** для понимания распределения титров в группах.
7. Для **графика истории** выберите тип данных с помощью радиокнопок (например, средние титры по возрастам).
8. В таблице статистики группы ознакомьтесь со сводными данными: количеством, средними значениями и т.д.
9. Если данные изменились, нажмите **"Перечитать"** для обновления.
10. Сохраните анализ в базу данных, нажав **"Сохранить"**.
11. Для создания отчета нажмите **"Печать"**, выберите формат (PDF или Excel) и место сохранения. Отчет будет включать таблицы, графики и статистику.
12. **"Сохранить все"** позволяет экспортировать полные данные анализа в отдельный файл.

## Советы

- Графики помогают выявить тенденции и аномалии в данных.
- Экспортированные отчеты подходят для презентаций и архивации.
- Если группы не отображаются, проверьте настройки анализа или перечитайте данные.

После анализа вы можете вернуться к главной форме или выполнить новый анализ.
