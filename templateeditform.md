# Редактирование шаблона

Форма редактирования шаблона позволяет создавать и изменять шаблоны пластин, определяющие размещение образцов для анализов.

## Интерфейс

- **Поле названия шаблона:** Введите уникальное имя для шаблона.
- **Таблица данных шаблона:** Список образцов с колонками: Ключ (Key), Тип (Type), Имя образца (CaseName), Возраст (Age), Количество (Count), Родительская платформа (ParentPlatform), Родительский возраст (ParentAge), Номер (Number), Комментарий (Comment). Редактируемые поля: CaseName, Age, Count, ParentPlatform, ParentAge, Number, Comment.
- **Визуализация платы:** Графическое представление 96-луночной платы с цветными метками типов образцов (Pos, Neg, Data и т.д.).
- **Кнопки управления строками:**
  - **Новый:** Добавляет новую строку образца в конец таблицы.
  - **Вставить:** Вставляет новую строку перед выбранной.
  - **Удалить:** Удаляет выбранную строку после подтверждения.

## Как создать или редактировать шаблон

1. Откройте редактор шаблона: нажмите "Создать" или "Редактировать" в форме шаблонов.
2. Введите название шаблона в соответствующее поле.
3. Заполните таблицу данных:
   - Нажмите **"Новый"** для добавления образца.
   - Или выберите существующую строку и нажмите **"Вставить"** для добавления перед ней.
   - Заполните поля: имя образца, возраст, количество, платформы, номер, комментарий.
4. Просматривайте визуализацию платы — она автоматически обновляется при изменениях в таблице, показывая размещение типов образцов.
5. Для удаления образца выберите строку в таблице и нажмите **"Удалить"**.
6. При закрытии формы программа проверит: наличие обязательных типов (Pos/Neg/Data), уникальность имен образцов, заполненность ключевых полей.
7. Если проверки пройдены, изменения сохранятся в базе данных.

## Советы

- Шаблон должен включать контрольные образцы (Pos — положительный, Neg — отрицательный) для калибровки.
- Визуализация помогает убедиться в правильном размещении на плате.
- Изменения в шаблоне влияют на будущие анализы, но не на уже выполненные.
- Сохраняйте шаблоны с понятными именами для легкого выбора.

После сохранения шаблон станет доступен в списке для использования в новых анализах.
