# Группы титров

Форма настройки групп титров позволяет установить пороги для разделения результатов анализов на группы по уровням титров.

## Интерфейс

- **Поля порогов титров:** 20 текстовых полей (Titer 1 до Titer 20) для ввода числовых значений порогов. Поля должны содержать возрастающие значения.
- **Кнопка "Сохранить":** Применяет изменения и сохраняет пороги в базу данных для выбранного ассая.

## Как настроить группы титров

1. Откройте форму групп титров: в форме шаблонов выберите ассай и нажмите кнопку **"Титры"**.
2. Форма загрузит текущие пороги для выбранного ассая.
3. Введите новые значения в поля Titer 1 — Titer 20. Значения должны быть в логическом порядке (например, 10, 50, 100 и т.д.), определяя границы групп.
4. Нажмите **"Сохранить"** для применения изменений к ассаю.
5. Пороги будут использоваться при расчете и классификации титров в результатах анализов.

## Советы

- Пороги делят титры на группы: например, Titer 1-10 — низкий, 11-50 — средний, 51+ — высокий.
- Изменения влияют только на будущие анализы, не на уже сохраненные результаты.
- Оставьте поля пустыми, если не нужно столько групп.
- Консультируйтесь с методикой анализа для правильных порогов.

После сохранения закройте форму. Пороги обновятся для выбранного ассая.
