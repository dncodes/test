# Руководство пользователя XemaScan

Добро пожаловать в XemaScan — приложение для проведения и анализа иммуноферментных анализов (ИФА). Это руководство поможет вам освоить основные функции программы.

## Начало работы

1. **Запустите приложение XemaScan.**
2. **Выберите версию:** Приложение имеет две версии — CVO и Xema. Выберите подходящую в зависимости от вашего устройства.
3. **Подключите устройство:** Убедитесь, что анализатор подключен и настроен в разделе настроек.
4. **Начните анализ:** Создайте новый анализ или откройте существующий.

## Основные задачи

- **Создание нового анализа:** Нажмите "Новый" на главной форме, выберите шаблон и начните сканирование.
- **Просмотр результатов:** Используйте "Поиск" для нахождения сохраненных анализов, затем просмотрите данные и графики.
- **Экспорт отчетов:** Сохраните результаты в PDF или Excel для дальнейшего использования.
- **Управление данными:** Редактируйте ассаи, шаблоны и настройки по мере необходимости.
