# Поиск результатов

Форма поиска позволяет найти и открыть сохраненные результаты анализов по различным критериям, таким как дата, тип анализа и параметры групп.

## Интерфейс

- **Поля дат:** Выберите начальную дату ("С") и конечную дату ("По") для ограничения периода поиска.
- **Фильтры по параметрам:** Выпадающие списки для выбора группы, единицы измерения, возраста, номера, платформы и других критериев. Эти фильтры сужают поиск.
- **Таблица ассаев:** Список доступных тестов (ассай). Выберите нужный ассай для поиска связанных результатов.
- **Поля ввода:** Текстовые поля для ввода номера и возраста (только цифры).
- **Кнопка "Поиск":** Выполняет поиск по выбранному ассаю и установленным фильтрам, открывает результаты.
- **Кнопка "Сегодня":** Автоматически устанавливает даты на текущий день и выполняет поиск за сегодня.
- **Кнопка "Все":** Показывает все сохраненные результаты без применения фильтров.

## Как выполнить поиск результатов

1. Откройте форму поиска, нажав "Открыть" или "Поиск" на главной форме.
2. Установите диапазон дат: выберите начальную и конечную даты.
3. В таблице ассаев выберите нужный тест (ассай). Если не выбрать, поиск будет по всем.
4. При необходимости настройте дополнительные фильтры: выберите группу, возраст, номер и другие параметры из выпадающих списков. Фильтры автоматически обновляются в зависимости от выбора.
5. Нажмите кнопку **"Поиск"**.
6. Результаты поиска откроются в новой форме, где вы сможете просмотреть и выбрать конкретный анализ.

## Советы

- Для быстрого поиска результатов за текущий день нажмите **"Сегодня"** — даты установятся автоматически.
- Чтобы увидеть все сохраненные анализы без фильтров, нажмите **"Все"**.
- Если фильтры не дают результатов, попробуйте расширить диапазон дат или снять некоторые ограничения.

После выбора результата в форме результатов поиска нажмите **"OK"** для просмотра детальных данных.
