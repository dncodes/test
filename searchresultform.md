# Результаты поиска

Форма результатов поиска отображает найденные анализы в удобном виде и позволяет просмотреть, экспортировать или удалить их.

## Интерфейс

- **Таблица результатов:** Группированная таблица с колонками: Ассай (тип теста), Дата, Имя группы, Номер, Возраст. Анализы сгруппированы по ScanId с заголовками и количеством записей в каждой группе.
- **Кнопка "OK":** Открывает выбранный анализ в форме детального просмотра результатов. Требует выбора хотя бы одной строки.
- **Кнопка "Сохранить в XLSX":** Экспортирует выбранные анализы в файл Excel или PDF. Открывает диалог для выбора формата и места сохранения.
- **Кнопка "Удалить":** Удаляет выбранные анализы из базы данных после подтверждения.
- **Кнопка "Сохранить все":** Сохраняет все отображенные результаты в файл.

## Как работать с найденными результатами

1. После поиска результаты автоматически загружаются в таблицу, сгруппированные по анализам.
2. Просмотрите список: каждая группа содержит информацию о дате, типе анализа и количестве образцов.
3. Выберите нужный анализ: кликните на строку или группу (для множественного выбора удерживайте Ctrl).
4. Нажмите **"OK"** для открытия детального просмотра выбранного анализа. Если ничего не выбрано, появится предупреждение.
5. Для экспорта данных нажмите **"Сохранить в XLSX"**, выберите формат файла (Excel или PDF) и укажите путь сохранения. Файл будет содержать таблицы, графики и статистику.
6. Чтобы удалить ненужные анализы, выберите их и нажмите **"Удалить"**. Подтвердите действие в диалоговом окне.
7. Для быстрого сохранения всех результатов нажмите **"Сохранить все"** и выберите место для файла.

## Советы

- Группировка помогает быстро найти нужный анализ по дате или типу.
- Экспортированные файлы можно использовать для отчетов или дальнейшего анализа в других программах.
- Удаление анализов необратимо, поэтому проверяйте выбор перед подтверждением.

После открытия анализа в форме результатов вы сможете просмотреть графики, статистику и детальные данные.
