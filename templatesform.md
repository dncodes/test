# Шаблоны

Форма управления шаблонами пластин и ассаями позволяет создавать, редактировать и удалять шаблоны для анализов, а также управлять тестами (ассаями).

## Интерфейс

- **Список шаблонов:** Список доступных шаблонов. Двойной клик на шаблоне открывает форму результатов для просмотра данных.
- **Таблица ассаев:** Список тестов с их параметрами.
- **Кнопки управления шаблонами:**
  - **Создать:** Открывает редактор для создания нового шаблона.
  - **Редактировать:** Открывает редактор для изменения выбранного шаблона.
  - **Удалить:** Удаляет выбранный шаблон после подтверждения.
  - **Копировать:** Создает копию выбранного шаблона с новым именем.
- **Кнопки управления ассаями:**
  - **Титры:** Открывает форму настройки групп титров для выбранного ассая.
  - **Редактор ассаев:** Открывает редактор ассаев для просмотра и изменения.
  - **Создать ассай:** Создает новый ассай.
  - **Изменить:** Открывает редактор ассаев (доступно только в админ режиме).
  - **Импорт:** Открывает форму импорта ассаев из файла.

## Как работать с шаблонами

1. Откройте форму шаблонов, например, из главной формы.
2. Просмотрите список шаблонов. Дважды кликните на шаблоне для открытия результатов анализа (если данные существуют).
3. Для создания нового шаблона нажмите **"Создать"**. Откроется редактор шаблонов.
4. Выберите шаблон в списке и нажмите **"Редактировать"** для внесения изменений.
5. Чтобы скопировать шаблон, выберите его и нажмите **"Копировать"**. Введите новое имя.
6. Для удаления шаблона выберите его и нажмите **"Удалить"**. Подтвердите действие.

## Как работать с ассаями

1. В таблице ассаев выберите нужный тест.
2. Нажмите **"Титры"** для настройки порогов групп титров для выбранного ассая.
3. Кнопка **"Редактор ассаев"** открывает полную форму редактирования ассаев.
4. **"Создать ассай"** добавляет новый тест в список.
5. **"Изменить"** позволяет редактировать параметры ассая (требуется админ доступ).
6. **"Импорт"** открывает форму для загрузки ассаев из .xasf файла.

## Советы

- Шаблоны определяют layout пластин и используются при создании новых анализов.
- Изменения в ассаях влияют на расчеты титров во всех анализах.
- После создания или изменения шаблона/ассая списки обновляются автоматически.
- Удаление шаблона или ассая необратимо, поэтому проверяйте перед подтверждением.

Форма служит центральным местом для подготовки к анализам.
