# Обновления

Форма обновлений позволяет проверить наличие новых версий программы и импортировать данные из предыдущих версий.

## Интерфейс

- **Метка версии:** Показывает номер текущей установленной версии XemaScan.
- **Кнопка "Обновления":** Открывает веб-браузер и переходит на страницу официального сайта для загрузки новых версий.
- **Кнопка "Импорт V1":** Импортирует базу данных из старой версии программы (файл hema.sqlite) после подтверждения. Текущая база очищается и заполняется конвертированными данными.

## Как проверить и установить обновления

1. Откройте форму обновлений, нажав "Обновления" в настройках.
2. Ознакомьтесь с текущей версией в метке.
3. Нажмите кнопку **"Обновления"** — программа откроет браузер на странице загрузки.
4. На сайте проверьте наличие новой версии и скачайте установщик, если доступен.
5. Запустите установщик и следуйте инструкциям по обновлению.
6. После установки перезапустите программу.

## Как импортировать данные из старой версии

1. Убедитесь, что у вас есть файл базы данных hema.sqlite из предыдущей версии XemaScan.
2. В форме обновлений нажмите **"Импорт V1"**.
3. В диалоговом окне подтвердите импорт. Программа предупредит, что текущая база будет заменена.
4. Выберите файл hema.sqlite, если потребуется.
5. Дождитесь завершения импорта — данные будут конвертированы в новый формат и загружены.
6. После импорта все анализы, шаблоны и настройки из V1 станут доступны.

## Важные предупреждения

- Импорт из V1 полностью заменяет текущую базу данных — сделайте резервную копию, если есть важные данные.
- Обновления могут добавить новые функции или исправить ошибки; регулярно проверяйте наличие обновлений.
- Если импорт не удался, проверьте файл hema.sqlite на целостность.

Форма помогает поддерживать программу в актуальном состоянии и переносить данные при обновлении.
