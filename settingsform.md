# Настройки

Форма настроек позволяет настроить подключение к устройству анализатора, параметры связи и общие параметры приложения.

## Интерфейс

- **Выпадающий список устройства:** Выберите модель анализатора (например, Multiskan, Biosan). При изменении обновляется доступность калибровки.
- **Выпадающий список порта:** Выберите COM-порт для подключения устройства.
- **Выпадающий список скорости:** Выберите скорость передачи данных (от 1200 до 128000 бод).
- **Выпадающий список стоп-битов:** Выберите количество стоп-битов (1, 1.5 или 2).
- **Выпадающий список бит данных:** Выберите количество бит данных (от 4 до 8).
- **Выпадающий список четности:** Выберите тип контроля четности (Чет, Нечет, Нет и т.д.).
- **Выпадающий список языка:** Выберите язык интерфейса (русский или английский).
- **Кнопка "Сохранить":** Применяет настройки, подключает устройство и обновляет статус калибровки.
- **Кнопка "Сброс":** Восстанавливает настройки к значениям по умолчанию.
- **Кнопка "Обновления":** Открывает форму проверки и установки обновлений.
- **Кнопка "Калибровка":** Запускает процесс калибровки устройства (если поддерживается), с индикацией прогресса.

## Как настроить устройство и приложение

1. Откройте настройки, нажав кнопку **"Настройки"** на главной форме.
2. Выберите модель устройства из списка. Это влияет на доступные функции, такие как калибровка.
3. Укажите COM-порт, к которому подключено устройство (обычно COM1-COM10).
4. Настройте параметры последовательного порта: скорость передачи, стоп-биты, биты данных и четность. Используйте значения, рекомендованные для вашего устройства.
5. Выберите язык интерфейса (ru для русского, en для английского).
6. Нажмите **"Сохранить"** для применения изменений. Приложение попытается подключиться к устройству и обновит статус.
7. Если устройство поддерживает калибровку, нажмите **"Калибровка"**. Дождитесь завершения процесса (может занять несколько минут).
8. Для возврата к заводским настройкам нажмите **"Сброс"**.
9. Кнопка **"Обновления"** открывает отдельную форму для проверки наличия новых версий программы.

## Советы и предупреждения

- Перед изменением настроек убедитесь, что устройство выключено или не используется.
- Если подключение не удается, проверьте правильность выбранного порта и параметров.
- Калибровка важна для точности измерений; выполняйте ее регулярно.
- Изменение языка вступит в силу после перезапуска приложения — программа предупредит об этом.
- Сохраняйте настройки перед закрытием формы, чтобы изменения применились.

После настройки закройте форму и вернитесь к работе с анализами.
