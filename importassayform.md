# Импорт ассаев

Форма импорта ассаев позволяет загрузить тесты (ассай) из внешнего файла .xasf в базу данных программы.

## Интерфейс

- **Кнопка "Открыть":** Открывает диалог выбора файла .xasf. После выбора файл распаковывается, и содержащиеся ассаи загружаются в таблицу для просмотра.
- **Кнопка "Импорт":** Импортирует отмеченные ассаи в базу данных после подтверждения. Существующие ассаи с тем же кодом заменяются.
- **Таблица ассаев:** Список ассаев из файла с чекбоксами "Импорт". Все ассаи отмечены по умолчанию.

## Как импортировать ассаи

1. Откройте форму импорта, например, нажав "Импорт" в редакторе ассаев.
2. Нажмите кнопку **"Открыть"** и выберите файл .xasf, содержащий ассаи.
3. Программа обработает файл и отобразит список ассаев в таблице.
4. Просмотрите ассаи: снимите галочки с тех, которые не нужно импортировать.
5. Нажмите **"Импорт"** для загрузки выбранных ассаев в базу данных.
6. В диалоговом окне подтвердите импорт. Существующие ассаи будут заменены новыми.

## Советы

- Файл .xasf должен быть корректным и содержать сериализованные ассаи.
- Импорт полностью заменяет существующие ассаи, поэтому сделайте резервную копию, если необходимо.
- После импорта новые ассаи появятся в списке редактора ассаев и смогут использоваться в шаблонах.

Форма закроется автоматически после завершения импорта. Если возникнут ошибки, проверьте файл и повторите попытку.
