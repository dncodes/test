# Редактор ассаев

Форма редактора ассаев позволяет создавать, редактировать и удалять тесты (ассай), используемые в анализах.

## Интерфейс

- **Список ассаев:** Список доступных тестов. Выберите ассай для загрузки его параметров в поля редактирования.
- **Поля редактирования:**
  - Код ассая (уникальный идентификатор)
  - Название ассая (описательное имя)
  - Параметр SP (параметр расчета)
  - Параметр CP (параметр расчета)
  - Параметр CN (параметр расчета)
  - Константа T1 (первая константа титра)
  - Константа T2 (вторая константа титра)
- **Кнопка "Добавить":** Создает новый пустой ассай для заполнения.
- **Кнопка "Удалить":** Удаляет выбранный ассай из списка после подтверждения.
- **Кнопка "Импорт":** Открывает форму импорта ассаев из файла.

## Как управлять ассаями

1. Откройте редактор ассаев, например, из формы шаблонов или меню.
2. В списке выберите существующий ассай — его параметры автоматически загрузятся в поля для редактирования.
3. Внесите изменения в поля: код, название или расчетные параметры (SP, CP, CN, T1, T2).
4. Изменения сохраняются при выборе другого ассая или закрытии формы.
5. Для создания нового ассая нажмите **"Добавить"**. Заполните поля кодом, названием и параметрами, затем закройте форму для сохранения.
6. Чтобы удалить ассай, выберите его в списке и нажмите **"Удалить"**. Подтвердите действие в диалоговом окне.
7. Кнопка **"Импорт"** открывает форму для загрузки ассаев из .xasf файла. Вы можете выбрать, какие ассаи импортировать.

## Советы

- Код ассая должен быть уникальным и использоваться в шаблонах.
- Параметры SP, CP, CN и константы T1/T2 влияют на точность расчетов титров — консультируйтесь с документацией или специалистом перед изменением.
- Удаление ассая может повлиять на существующие анализы и шаблоны, поэтому проверяйте зависимости.
- Импорт полезен для переноса ассаев между установками программы.

После завершения редактирования закройте форму. Изменения применятся автоматически.
